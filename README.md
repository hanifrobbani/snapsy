## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

Snapsy is a simple tools that help you catch snapshot from several populer chain, you can take a free snapshot from token or an NFTs with many popular chain! just paste the contract address & get the data for free with clean & smooth ui.

Main feature:
* A simple & clean ui 
* Diagram and analytic tools to help you manage the data more easier
* Download the data as CSV or TXT for free!

## Want to Contribute?

This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hanifrobbani/snapsy.git
   ```
2. Install packages
   ```sh
   bun install #recommended use bun
   npm install 
   ```
3. Change git remote url to avoid accidental pushes to base project
   ```sh
   git remote set-url origin hanifrobbani/snapsy
   git remote -v # confirm the changes
   ```
4. Change branch
   ```sh
   git checkout -b dev/your-feature # wait for approval
   ```
